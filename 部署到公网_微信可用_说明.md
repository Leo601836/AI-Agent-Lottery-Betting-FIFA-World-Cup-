# 把面板放到公网 · 真正在微信里打开/分享

> 本地双击、`启动面板_*`（localhost）、`手机微信扫码访问_*`（同一 WiFi 手机/微信扫码）已能用。
> 但要做到**任意网络、发给好友、在微信里直接点开**，需要把网页放到公网（https）。下面两种都不用写代码，选一种即可。
> 已为你准备好根入口 `index.html`（自动跳转到面板），所以无论用哪种方式，访问网站根地址就能直接进面板。

---

## 方法一：Netlify Drop（最快，2 分钟，免注册即可先试）

1. 浏览器打开 **https://app.netlify.com/drop**
2. 把整个 **FIFA World Cup 文件夹**（至少包含 `index.html`、`index.html`、`世界杯AI助手.js`）直接**拖进**网页中间的虚线框。
3. 等几秒，页面会给你一个公网网址，形如 `https://随机名.netlify.app`。
4. 这个 https 网址**可以在微信里直接打开/转发**：复制它，发到微信聊天框或收藏，点开即用。
5. （可选）注册免费账号后可改成好记的名字、并支持以后覆盖更新。

> 提示：每次内容更新后，重新拖一次文件夹即可获得新版本（同一账号下可保持同一网址）。

---

## 方法二：GitHub Pages（永久免费、网址固定，适合长期用）

1. 注册/登录 **https://github.com**，点右上角 **+ → New repository**，名字随意（如 `worldcup`），选 **Public**，创建。
2. 进入仓库页 → **Add file → Upload files**，把 `index.html`、`index.html`、`世界杯AI助手.js`（以及需要的 json/xlsx）拖上去 → **Commit changes**。
3. 仓库 **Settings → Pages → Build and deployment → Source 选 “Deploy from a branch”**，Branch 选 `main` / 根目录 `/ (root)` → Save。
4. 等 1–2 分钟，页面顶部会出现网址 `https://你的用户名.github.io/worldcup/`。
5. 这个 https 网址即可在**微信内打开与转发**。以后更新：重新 Upload files 覆盖即可。

---

## 关于「微信里打开」的说明（重要）

- **公网 https 链接**（上面两种）在微信内浏览器可直接打开、可转发给好友——这是最稳的方式。
- **localhost / 局域网（手机扫码）** 仅限你自己、且手机要与电脑同一 WiFi；微信对非 https、未备案链接有时会拦截或提示「在浏览器打开」，所以**对外分享请用公网方式**。
- 面板是**纯前端单页**，放到任何静态托管（Netlify / GitHub Pages / Vercel / Cloudflare Pages / 自己的服务器）都能跑，无需后端。
- 右下角 AI 助手（`世界杯AI助手.js`）默认走免 Key 的 Puter·GLM，公网部署同样可用。
